### ✅ 任务 2：MongoDB 索引创建脚本

**文件**：`backend/database/indexes.py`

**实现内容**：
- `create_indexes()` - 创建所有索引
- `list_indexes()` - 列出集合索引
- `drop_index()` - 删除指定索引
- `verify_indexes()` - 验证索引状态
- `recreate_all_indexes()` - 重新创建所有索引

**索引列表**：
| 集合 | 索引 | 类型 |
|------|------|------|
| users | openid | 唯一 |
| users | user_id | 唯一 |
| users | created_at | 普通 |
| sessions | session_id | 唯一 |
| sessions | user_id | 普通 |
| sessions | user_id + updated_at | 复合 |
| tokens | token | 唯一 |
| tokens | user_id | 普通 |
| tokens | expires_at | 普通 |
| tokens | user_id + is_revoked | 复合 |
| spaces | id | 唯一 |
| spaces | user_id | 普通 |

### ✅ 任务 6：JWT Token 认证实现

**文件**：`backend/auth.py`

**实现内容**：
- `create_access_token()` - 创建访问令牌
- `create_refresh_token()` - 创建刷新令牌
- `decode_token()` - 解码 Token
- `verify_token()` - 验证 Token 依赖
- `get_current_user()` - 获取当前用户
- `AuthService` 类 - 认证服务

**Token 结构**：
```python
{
  "sub": "user_id",
  "openid": "wx_openid",
  "exp": 1738550400,
  "iat": 1738464000,
  "type": "access"  # or "refresh"
}
```

## 五、使用指南

### 1. 安装依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，填写必要的配置
```

### 3. 创建 MongoDB 索引

```bash
python -m backend.database.indexes create
```

### 4. 启动服务

```bash
python -m backend.main
```

### 5. 测试 API

```bash
# 健康检查
curl http://localhost:8000/health

# 数据库健康检查
curl http://localhost:8000/health/db

# 微信登录（需要微信 code）
curl -X POST http://localhost:8000/api/auth/wechat/login \
  -H "Content-Type: application/json" \
  -d '{"code": "your_wechat_code", "device_type": "miniprogram"}'
```
