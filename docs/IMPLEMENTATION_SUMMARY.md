## 功能特性

### 隐含关联问题识别

支持识别 10 种隐含关联问题：

| 问题类型 | 示例 |
|---------|------|
| 挂科与奖学金 | "挂科对奖学金有影响吗？" |
| 软著加分 | "软著可以加分吗？" |
| 专利加分 | "专利能加分吗？" |
| 竞赛加分 | "竞赛获奖怎么加分？" |
| 转专业 | "转专业需要什么条件？" |
| 延期毕业 | "延期毕业有什么影响？" |
| 重修费用 | "重修要交费吗？" |
| 补考绩点 | "补考对绩点有影响吗？" |
| 处分评优 | "处分影响评优吗？" |
| 宿舍处分 | "宿舍违规会受处分吗？" |
